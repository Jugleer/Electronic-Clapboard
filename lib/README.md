# Local Libraries

PlatformIO auto-discovers library folders placed here. Use this for code that is modular enough to be a library (with its own `library.json` or `library.properties`) but not yet worth publishing externally.

For project source that isn't library-shaped, keep it in [`../src`](../src) instead.

See the [PlatformIO library docs](https://docs.platformio.org/en/latest/librarymanager/index.html) for details.
